CollabFlow est une application web moderne de gestion de projets collaboratifs, inspirée de Trello. Elle permet aux équipes de gérer leurs projets, tâches et utilisateurs avec une interface intuitive et réactive.

## 🚀 Fonctionnalités principales

- **Dashboard interactif** : Statistiques en temps réel, graphiques d'avancement, activités récentes
- **Gestion de projets** : CRUD complet, filtres, barre de progression
- **Kanban Board** : Vue Trello-like avec drag & drop, colonnes personnalisables
- **Gestion des tâches** : Priorités, statuts, échéances, assignation
- **Système d'authentification** : JWT, sessions, rôles (Admin/Utilisateur)
- **Multilingue** : Support Français, Anglais, Arabe avec RTL
- **Dark Mode** : Thème clair/sombre persistant
- **CRUD complet** : Projets, Tâches, Utilisateurs, Rôles, Étiquettes

## 🛠️ Stack technique

**Frontend** : HTML5, CSS3, JavaScript, Chart.js, SortableJS  
**Backend** : Node.js, Express.js, MongoDB, JWT, bcrypt  
**API** : RESTful avec architecture MVC

## 📊 Statistiques

- 100% responsive
- Design moderne avec variables CSS
- Compatible tous navigateurs
